## Prerequisites:
- installed [nvm](https://github.com/nvm-sh/nvm) ([for windows](https://github.com/coreybutler/nvm-windows))

## Install:
```
./nvm-use.sh #for linux just nvm use
npm install
```

## Start
```
npm run start // Navigate to `http://localhost:4200/`
```
